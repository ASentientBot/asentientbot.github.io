Create Projector workaround

Adobe disabled the Create Projector option in Flash Player 11.3 and newer, so I partially reimplemented it a couple years ago. These tools combine SWF files with Flash Player projectors to produce standalone macOS and Windows apps.



download

Click here to download my tools and source code. There is an AppleScript implementation for macOS, and a Flash-based one for Windows. I am unlikely to release any updates.
