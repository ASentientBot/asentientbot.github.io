fixing Neat NM-1000 driver

The Neat NM-1000 driver is incompatible with recent macOS versions but can be patched.

solution

The package fails harmlessly, but these commands are necessary to fix the driver:
sudo install_name_tool -change /System/Library/Frameworks/Carbon.framework/Versions/A/Carbon /System/Library/Frameworks/ICADevices.framework/Versions/A/ICADevices '/Library/Image Capture/Devices/NeatScannersICDriver.app/Contents/Frameworks/Neat.framework/Versions/A/Neat' sudo codesign -fs - '/Library/Image Capture/Devices/NeatScannersICDriver.app/Contents/Frameworks/Neat.framework/Versions/A/Neat'
After a reboot, the scanner is detected by Image Capture, though it sometimes takes a few tries to communicate for some reason...
